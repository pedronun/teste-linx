# Linx - Feito por Pedro Lucas

## Tecnologias usadas

- ReactJs
- Typescript
- Sass

Para rodar o projet no localhost execute os seguinte códigos:

Primeiramente clone o repositório para a máquina local e digite no prompt

```sh
cd linx
yarn install
yarn start
```
